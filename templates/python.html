¡Instalando Python!

Linux: Sistemas operativos como Linux traen el Python 2.7.x de forma prediterminada, para probar si en verdad esta instalado, puede abrir la consola y escribir python, esto le mostrará si Python está instalado en su computadora

Mac/Windows: Descargue Python-2.7.7, ejecútelo y siga los pasos de instalación del programa


¡Instalando Flask!

Flask es el mini-framework utilizado en Andrei's Web Desing, por lo que es necesario para ejecutar el programa

Si esta en Linux/Mac puede utilizar el comando:
$ sudo pip install Flask

En Windows, puede ejecutar el comando en la cmd con permisos de administrador:
$ pip install Flask
En caso de que no tenga el comando pip, puede leer la siguiente guía para instalarlo.
